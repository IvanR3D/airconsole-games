# AirConsole Games - Fundacion STEAM RD

Juegos desarrollados para usar sobre la plataforma [AirConsole](https://www.airconsole.com/).

## Juegos Disponibles

### STEAM Platformer
Juego de plataformas multijugador (1-4 jugadores). Carrera a la meta mientras colectas monedas y evitas obstaculos.

**Enlaces para jugar:**
- **Pantalla (TV/PC):** https://ivanr3d.github.io/airconsole-games/platformer/screen.html
- **Control (Telefono):** https://ivanr3d.github.io/airconsole-games/platformer/controller.html

---

### STEAM Trivia
Juego de trivia multijugador con preguntas de diferentes categorias: Ciencia, Matematicas, Robotica, Quimica, Tecnologia, Historia y Geografia.

**Enlaces para jugar:**
- **Pantalla (TV/PC):** https://ivanr3d.github.io/airconsole-games/trivias/screen.html
- **Control (Telefono):** https://ivanr3d.github.io/airconsole-games/trivias/controller.html
